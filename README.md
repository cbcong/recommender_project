# recommender_project

## 改进策略与动机（准确性 × 多样性，面向 SCI 一区质量）

- **多目标训练（Multi-Objective）**：在主准确度损失（MSE / BPR）之外，显式加入覆盖度、ILAD、类别分布均衡等辅助损失，并通过可调权重或 MGDA/Pareto 策略动态寻优。这样可在精确度与新颖度之间形成稳定的折中，而非单一指标的局部最优。
- **知识蒸馏 + 重排序**：使用 MMR、DPP 等确定性多样性算法产生教师信号（概率标签或 pairwise 目标），学生模型学习兼顾相关性与多样性的排序分布，推理时无需昂贵的贪心解码即可输出“多样性友好”的 Top-K。
- **对比学习与邻域相似性正则**：以内容-协同对齐为基底，增加物品/用户层面的对比任务或邻域一致性损失，强化长尾物品表示与多兴趣用户的区分度，缓解“热门吸引”现象。
- **可学习 & 用户自适应流行度惩罚**：在 HybridTail 中将固定的流行度惩罚改为可训练参数，并结合用户历史流行度分布自适应缩放，自动平衡热门抑制与准确率损失，兼顾冷启动用户的公平性。
- **注意力/门控融合**：在 HybridNCF 协同-内容融合处引入多头注意力或门控，让模型按用户状态自适应分配协同特征与内容特征的权重，避免“一刀切”融合策略。
- **多通道候选与聚合**：利用多子模型或多次随机采样生成异质候选集，再经 MMR/投票/重排序融合，显著提升覆盖度与类别丰富度。
- **超参与可解释评估**：在调参脚本中网格搜索多目标权重、对比温度、流行度惩罚强度等关键超参，并在评估阶段同时报告 Recall/NDCG、Coverage、LongTailShare、Novelty（可拓展 ILAD/Category Coverage），以多维度展示改进成效。
