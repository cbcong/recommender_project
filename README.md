# recommender_project

## 改进策略与动机（准确性 + 多样性）

- **多目标训练（Multi-Objective）**：在主准确度损失（如 MSE/BPR）之外，引入覆盖度、ILAD 或类别分布的惩罚/辅助损失，并通过权重组合平衡精确度与新颖度。动机是避免单一准确性目标导致的同质化推荐。
- **知识蒸馏与重排序**：用确定性多样性算法（MMR、DPP 等）生成教师得分/标签，学生模型学习多样性概率或排序分布，在推理端可直接输出兼顾准确与多样的排序，省去贪心选取开销。
- **对比学习与邻域相似性**：在内容与协同信号对齐的基础上，引入对比任务（物品/用户正负样本）或邻居相似性损失，强化长尾物品和多兴趣用户的表示，提升覆盖率与新颖度。
- **可学习的流行度惩罚**：在 HybridTail 中将固定的流行度惩罚改为可训练或用户自适应形式，动态抑制热门偏置，同时减少对整体准确率的副作用。
- **注意力或门控融合**：在 HybridNCF 的协同/内容融合处加入注意力或门控，使模型为不同用户自适应分配两类特征的权重，提升个性化表达能力。
- **多通道候选与聚合**：使用多子模型或多次采样生成差异化候选，再通过 MMR/投票聚合，提升最终集合的覆盖度和多样性。
- **Pareto 优化权重**：采用 MGDA 等方法自适应更新多目标梯度权重，无需手工设定固定 λ，直接沿 Pareto 前沿优化准确性与多样性。
